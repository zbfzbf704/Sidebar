## 上传到 GitHub

### 方法一：使用 GitHub Desktop

1. 下载并安装 [GitHub Desktop](https://desktop.github.com/)
2. 登录您的 GitHub 账户
3. 点击 "File" -> "New Repository"
4. 填写仓库信息：
   - Name: `Sidebar`
   - Description: `基于 ShareX 开发的 Windows 侧边栏应用程序`
   - 选择 "Public"（公开仓库，符合 GPL v3 要求）
   - 不要勾选 "Initialize this repository with a README"
5. 点击 "Create Repository"
6. 将本目录的所有文件拖拽到 GitHub Desktop
7. 填写提交信息：`Initial commit: Sidebar open source release`
8. 点击 "Commit to main"
9. 点击 "Publish repository"

### 方法二：使用 Git 命令行

1. 安装 [Git for Windows](https://git-scm.com/download/win)

2. 打开命令行，进入本目录：
```bash
cd "C:\Users\zbfzb\Documents\projects\Sidebar\开源协议上传完整代码"
```

3. 初始化 Git 仓库：
```bash
git init
```

4. 添加所有文件：
```bash
git add .
```

5. 创建初始提交：
```bash
git commit -m "Initial commit: Sidebar open source release"
```

6. 在 GitHub 上创建新仓库（访问 https://github.com/new）

7. 添加远程仓库并推送：
```bash
git remote add origin https://github.com/您的用户名/Sidebar.git
git branch -M main
git push -u origin main
```

### 方法三：使用 GitHub 网页上传

1. 访问 https://github.com/new 创建新仓库
2. 填写仓库信息（同上）
3. 创建仓库后，点击 "uploading an existing file"
4. 将本目录的所有文件拖拽到上传区域
5. 填写提交信息并提交

## 上传到 Gitee（码云）

### 使用 Gitee 网页上传

1. 访问 https://gitee.com/projects/new 创建新仓库
2. 填写仓库信息
3. 选择 "公开" 仓库
4. 创建仓库后，点击 "上传文件"
5. 将本目录的所有文件上传

### 使用 Git 命令行

```bash
cd "C:\Users\zbfzb\Documents\projects\Sidebar\开源协议上传完整代码"
git init
git add .
git commit -m "Initial commit: Sidebar open source release"
git remote add origin https://gitee.com/您的用户名/Sidebar.git
git push -u origin master
```
