## 使用说明

1. **配置 API 服务器**
   - 打开 `AuthAPIClient.cs`
   - 将 `AUTH_SERVER` 常量替换为您的 API 服务器地址

2. **更新邮箱地址（可选）**
   - 搜索所有 `your-email@example.com` 占位符
   - 替换为您的联系邮箱（如果需要）

3. **上传到代码托管平台**
   - 参考 `上传指南.md` 或 `README_上传说明.md`
   - 使用 `上传前检查清单.md` 进行最终验证
