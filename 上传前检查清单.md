## 📝 上传步骤

1. **选择上传平台**
   - GitHub（推荐）
   - Gitee（码云）
   - GitLab
   - SourceForge

2. **创建仓库**
   - 仓库名称：`Sidebar`
   - 描述：`基于 ShareX 开发的 Windows 侧边栏应用程序`
   - 选择 **Public**（必须公开，符合 GPL v3）

3. **上传文件**
   - 使用网页上传或 Git 命令行
   - 参考 `上传指南.md` 或 `README_上传说明.md`

4. **验证上传**
   - 检查所有文件是否已上传
   - 检查 README.md 是否正确显示
   - 检查 LICENSE 文件是否存在
