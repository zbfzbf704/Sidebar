# 隐私信息移除说明

## 概述

本开源代码已移除所有隐私和敏感信息，以符合开源协议要求并保护开发者隐私。

## 已移除/替换的信息

### 1. 邮箱地址
- **已替换为**：`your-email@example.com`
- **位置**：所有源代码文件中的版权声明和反馈信息

### 2. API 服务器地址
- **已替换为**：`https://your-api-server.com`
- **位置**：`AuthAPIClient.cs` 中的 `AUTH_SERVER` 常量

### 3. 其他敏感信息
- 所有硬编码的 API 密钥（如果有）
- 个人服务器配置信息
- 其他可能泄露隐私的信息

## 使用本代码

如果您要使用或修改本代码，请：

1. **配置 API 服务器**
   - 打开 `AuthAPIClient.cs`
   - 将 `AUTH_SERVER` 常量替换为您的 API 服务器地址
   - 配置必要的 API 密钥和认证信息

2. **更新邮箱地址**
   - 搜索所有 `your-email@example.com` 占位符
   - 替换为您的联系邮箱（如果需要）

3. **配置其他服务**
   - 检查代码中是否有其他需要配置的服务
   - 根据您的需求进行配置
